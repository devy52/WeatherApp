## Run Locally

Clone the project


```bash
  git clone https://github.com/devy52/WeatherApp
```

Go to the project directory

```bash
  cd WeatherApp
```

Install dependencies

```bash
  npm install (or) npm i
```

Start the server

```bash
  npm run start (or) npm start 
```
